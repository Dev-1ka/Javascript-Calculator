# Calculator Web Application

A simple and responsive calculator web application developed using HTML, CSS, and JavaScript to practice basic frontend development concepts and DOM manipulation.

## Features

- Basic arithmetic operations
- Interactive user interface
- Responsive design
- Button click functionality
- Real-time calculation display

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Purpose

This project was developed as a beginner-level mini project to understand fundamental concepts of web development, event handling, and JavaScript operations.
